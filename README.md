# MealApp

A meal planning application built with Vue 3, TypeScript, and Vite.

## Tech Stack

- **Vue 3** with Composition API (`<script setup>`)
- **TypeScript**
- **Vite** - Build tooling
- **Tailwind CSS v4** - Utility-first CSS
- **shadcn-vue** - UI component library
- **Vue Router 4** - Client-side routing

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
